# Documenting deployment types in Elastic Docs

Elastic supports multiple deployment types: self-managed, {{eck}}, {{ece}}, {{ech}}, and {{serverless-full}}. The same task, such as configuring a setting or setting up monitoring, can require different steps on different deployment types. Some features aren't available on every deployment type.

We document the {{stack}} across all deployment types in a single source of truth where possible, and handle variation explicitly when needed.

:::{note}
If you have questions about deployment types or how to document them, reach out to **@elastic/docs** in the related GitHub issue or PR.
:::
