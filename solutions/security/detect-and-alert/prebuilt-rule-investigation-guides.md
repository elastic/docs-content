# Prebuilt rules investigation guides

Browse investigation guides for Elastic [prebuilt detection rules](/solutions/security/detect-and-alert/prebuilt-rules.md) by MITRE ATT&CK tactic using the sidebar navigation.

You can also find guides through the [prebuilt rule catalog](/solutions/security/detect-and-alert/prebuilt-rule-catalog.md) — rules that include an investigation guide have a **Guide** link in the catalog table.

To learn more about what investigation guides contain and how to use them, refer to [About prebuilt rules](/solutions/security/detect-and-alert/about-prebuilt-rules.md#prebuilt-investigation-guides).
