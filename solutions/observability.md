# Elastic Observability solution overview

Elastic Observability provides unified observability across applications and infrastructure. It combines logs, metrics, application traces, user experience data, and more into a single, integrated platform — with AI-powered analysis to help you detect issues and resolve them faster.

|     |     |
| --- | --- |
| [Observe with AI](/solutions/observability/ai/observability-ai.md) | Use built-in AI tools — AI Assistant, Agent Builder, and LLM observability — to enhance workflows, answer questions, and solve problems. |
| [Observe your data](/solutions/observability/observe-your-data.md) | Search, visualize, and analyze your observability data across all signals using Discover, dashboards, and ES\|QL. |
| [Observe your application](/solutions/observability/applications/index.md) | Monitor application performance and real-time service behavior using APM and distributed tracing. |
| [Observe your infrastructure](/solutions/observability/infra-and-hosts.md) | Monitor metrics from hosts, containers, Kubernetes, and other infrastructure resources. |
| [Manage logs](/solutions/observability/logs.md) | Ingest, parse, and analyze log data at petabyte scale from any source. |
| [Manage incidents](/solutions/observability/incident-management.md) | Respond to incidents with alerting, cases, and service-level objectives (SLOs). |
| [Synthetic monitoring](/solutions/observability/synthetics/index.md) | Proactively simulate user journeys and API calls to test availability and functionality. |
| [Real user monitoring (RUM)](/solutions/observability/applications/user-experience.md) | Capture and analyze how real users interact with your web applications. |
| [Streams monitoring](/solutions/observability/streams/streams.md) | Manage and monitor Elasticsearch data streams with AI-powered parsing and field extraction. |
