Use the search bar on the **Cases** page to quickly find cases and case data. You can search for case titles, descriptions, and IDs. Keywords and text search is supported.

{applies_to}`stack: ga 9.3` You can also search for alert and event IDs, observable values, case comments, and file names. For example, you can search for a specific IP address that's been specified as an observable, a colleague's comment, or the ID of an alert that's attached to the case.
